# Running App POC

## About

This is a proof-of-concept project for a **territory mapping running app** — think Strava but with a territorial conquest twist. When you run through an area, that area becomes yours. The core gameplay loop is: run, claim territory, compete with others.

Beyond the base concept, the product vision extends further with additional features and mechanics to be explored.

## Project Structure

- `idea/` — Product ideas, feature brainstorms, and vision docs
- `competitors/` — Competitor analysis and market research
- `application-planning/` — Technical planning, architecture, and implementation roadmap

## Core Concept

- Territory mapping through running activity
- Area-based ownership — run over an area to claim it
- Gamification layer on top of fitness tracking
- Social/competitive elements

## Development Guidelines

- This is a POC — favour speed and clarity over perfection
- Document decisions and reasoning in the relevant folders
- Keep competitor research factual and up to date
- Every time you change or add anything in this project, create a commit and push it
